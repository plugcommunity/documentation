# plugMessage message

Event passed when a message gets broadcasted.

### Packet Example

```js
[{
    "a": "plugMessage", // Event name
    "p": "xxx",
    "s": "xxxx"         // Room name
}]
```
### Real life example
```js
[{
    "a": "plugMessage",
    "p": "test message",
    "s": "loves-kpop"
}]
```
