# userLeave message

Event passed when a user leaves.


### Packet Example

```js
{
    'a': 'userLeave',       // Event name
    'p': -1,                // ID of the user
    's': 'xxxx'             // Room name
}
```
### Real life example
```js
{
    'a': 'userLeave',
    'p': 2348789,
    's': 'loves-kpop'
}
```
