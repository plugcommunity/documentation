# grab message

Event passed when a user grabs a song.


### Packet Example

```js
{
    'a': 'grab',            // Event name
    'p': -1,                // ID of the user
    's': 'xxxx'             // Room name
}
```
### Real life example
```js
{
    'a': 'grab',
    'p': 2348789,
    's': 'loves-kpop'
}
```
