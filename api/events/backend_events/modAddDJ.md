# modAddDJ message

Event passed when a moderator adds a user to waitlist

### Example

```js
{
	"a": "modAddDJ", 			// Event name
	"p": {						
		"mi": 5107634, 			// Moderator ID
		"m": "TNC.Lizzie", 		// Moderator name
		"t": "Chocacho"			// User name
	}, 
	"s": "thenightcoreclub"		// Source event
}
```
