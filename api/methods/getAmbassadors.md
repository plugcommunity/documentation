# Get Ambassadors

Returns an array of the ambassadors in the community.

### Code

```js
API.getAmbassadors();
```

### Response

```js
[
    {
        id: 80,
        username: '',
        avatarID: '',
        language: 'en',
        joined: undefined,
        level: 1,
        role: 0,
        gRole: 0,
        badge: undefined,
    }
]
```
