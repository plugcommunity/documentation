# Get Audience

Returns an array containing the user objects of all of the users in the audience excluding the DJ.

### Code

```js
API.getAudience();
```

### Response

```js
[
    {
        id: 80,
        username: '',
        avatarID: '',
        language: 'en',
        joined: undefined,
        level: 1,
        role: 0,
        gRole: 0,
        badge: undefined,
    }
]
```
