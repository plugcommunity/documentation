# Get User

Returns the user object of the logged in user.

If you specify an ID, it returns the object of that user.

### Code

```js
API.getUser();
```

### Response

```js
{
    id: 80,
    username: '',
    avatarID: '',
    language: 'en',
    joined: undefined,
    level: 1,
    role: 0,
    gRole: 0,
    badge: undefined,
}
```
