# Get Staff

Returns an array of the staff in the room.

You can view the API Constants [here](/api/constants.md) and the in depth information on roles, [here](/api/roles.md)

### Code

```js
API.getStaff();
```

### Response

```js
[
    {
        id: 80,
        username: '',
        avatarID: '',
        language: 'en',
        joined: undefined,
        level: 1,
        role: 0,
        gRole: 0,
        badge: undefined,
    }
]
```
