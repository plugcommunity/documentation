# Get Wait List

Returns an ordered array of the users present in the wait list.

### Code

```js
API.getWaitList();
```

### Response

```js
[
    {
        id: 80,
        username: '',
        avatarID: '',
        language: 'en',
        joined: undefined,
        level: 1,
        role: 0,
        gRole: 0,
        badge: undefined,
    }
]
```
