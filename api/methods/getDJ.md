# Get DJ

Returns the current DJ, or `undefined` if there is no DJ.

### Code

```js
API.getDJ();
```

### Response

```js
{
    id: 80,
    username: '',
    avatarID: '',
    language: 'en',
    joined: undefined,
    level: 1,
    role: 0,
    gRole: 0,
    badge: undefined,
}
```
